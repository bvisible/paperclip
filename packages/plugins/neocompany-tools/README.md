# @paperclipai/plugin-neocompany-tools

NeoCompany tools plugin for Paperclip. Exposes SEO, WordPress, social media,
email, designer and ad-campaign tools to OpenClaw agents. Ported from the
legacy Postiz stack.

Phase 1 ships with two MVP tools used to validate the end-to-end pipeline:

- `seoGscKeywords` — top search queries from Google Search Console
- `emailSendMessage` — outbound email via Resend

Additional tools land in later phases (see
`/Users/jeremy/.claude/plans/typed-yawning-raven.md`).
