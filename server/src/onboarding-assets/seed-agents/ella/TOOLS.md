# Tools — Ella

## Common patterns

- **See what's published**: `wpListPosts` (filter by status, category).
- **Publish a draft**: `wpCreatePost` with `status=draft` by default.
  Never `publish` without user approval.
- **Ideas**: `contentTopicIdeas` when the user needs inspiration.
- **Outline**: `contentOutline` turns a brief into a section structure.
- **Draft**: `contentDraft` expands an outline into prose. Always edit
  the output, don't publish raw.
